
# QR code Generator
QR code Generator using Node.JS. Amazing and interesting thing that makes any thing unique itself.





## Run Locally

Clone the project

```bash
  git clone https://github.com/deepak640/QR-Generator.git
```

Go to the project directory

```bash
  cd QR-Generator
```

Install dependencies

```bash
  npm install
```

Start using

```bash
  node index.js
```
